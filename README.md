# Wakiso District Asset Management System

## Description
A web-based system for managing district assets using Flask and MySQL.

## Features
- Asset registration
- Asset listing
- Basic system navigation

## Technologies Used
- Python (Flask)
- MySQL
- HTML

## Setup Instructions
1. Install Python
2. Install requirements:
   pip install -r requirements.txt
3. Run the app:
   python app.py
